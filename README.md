# 📝 To-Do List App

Um aplicativo Android nativo, simples e intuitivo para gerenciamento de tarefas diárias. 
Este projeto foi desenvolvido focado no aprendizado prático de componentes de interface modernos e arquitetura no desenvolvimento mobile.

---

## 🚀 Tecnologias Utilizadas

* **Linguagem:** Kotlin
* **Interface UI:** Jetpack Compose 
* **Arquitetura:** MVVM (Model-View-ViewModel) 
* **Componentes de UI:** LazyColumn, TopAppBar, IconButton, além de gerenciamento de estados dinâmicos

---

## 📱 Funcionalidades

* [x] Visualização de tarefas em formato de lista fluida.
* [x] Adicionar e remover itens da lista em tempo real.
* [x] Marcar tarefas como concluídas.
* [x] Interface limpa.
